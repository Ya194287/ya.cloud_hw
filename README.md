# ya.cloud_hw
Витрина топ-2 товаров по выручке в разрезе городов (PySpark)

Проект для сбора витрины топ-2 товаров по выручке в разрезе городов на основе данных о пользователях, заказах и продуктах. Реализован на PySpark в Yandex Cloud Data Proc.

## Технологии
- Yandex Cloud Data Proc
- Apache Spark / PySpark
- Apache Zeppelin
- HDFS / S3 (Object Storage)

## Структура репозитория
- `Untitled Note 1_2MJWQPJT7.zpln` — экспорт ноутбука из Zeppelin с полным кодом решения

## Задачи, решённые в проекте
1. Расчёт revenue = qty × price для каждого заказа
2. Объединение таблиц users, orders, products
3. Агрегация метрик:
   - orders_cnt (количество заказов)
   - qty_sum (суммарное количество товаров)
   - revenue_sum (суммарная выручка)
4. Ранжирование товаров по выручке внутри каждого города с помощью оконных функций
5. Выбор Top-2 товаров для каждого города
6. Сохранение результатов в HDFS и S3
7. Проверка сохранённых данных чтением из обоих хранилищ

## Результат
Итоговая витрина содержит для каждого города два товара с наибольшей выручкой.

## Как использовать
1. Импортируйте файл `.zpln` в Zeppelin
2. Запустите ячейки последовательно
3. Результат сохранится в HDFS и S3
